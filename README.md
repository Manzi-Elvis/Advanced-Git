# Git Advanced

A repository dedicated to mastering Git from beginner to advanced concepts through hands-on exercises.

## Topics Covered

- Git History
- Amending Commits
- Interactive Rebase
- Squashing
- Splitting Commits
- Cherry Picking
- Branching
- Merging
- Rebasing
- Stashing
- Tags
- Merge Conflicts
- Remote Repositories
- Git Reflog

## Goal

By the end of this repository I should be comfortable using Git in professional software development teams and be able to confidently manipulate commit history, collaborate through branches, and recover from mistakes.
